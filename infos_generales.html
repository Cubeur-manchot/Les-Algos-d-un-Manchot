Les Informations Générales d'un Manchot

(un genre de pseudo-FAQ)

Bienvenue, visiteur !

Ce site sert principalement au stockage de mes algos. Ces algos sont triés par sets, naturellement. Ils sont tous accessibles depuis la page d'accueil.

Les sets entiers sont en blanc, tandis que les sets incomplets sont grisés.

En cliquant sur un set, la page correspondante s'ouvre dans un nouvel onglet du navigateur. Ainsi, vous ne perdez pas de vue la page d'accueil, qui permet d'accéder à toutes les pages du site ;)

S'il y a la moindre erreur sur mon site à propos d'algos, un bug technique ou une proposition d'amélioration, n'hésitez pas à m'envoyer un pigeon :

  • sur le discord francophone : par MP ou en me mentionnant dans #general_cubes (@Cubeur-manchot)
  • sur le forum Francocube : en cliquant sur le lien du forum Francocube depuis la page d'accueil, vous tomberez directement sur la page pour m'envoyer un MP
Les images de cube sont générées par VisualCube. Une vue plane est utilisée pour la plupart des sets. Une double-vue est utilisée pour le blind notamment, pour mieux voir tout autour du cube.

En cliquant sur l'image d'un cas, une animation alg.cubing de l'algo s'ouvre dans un nouvel onglet.

Pour info, je ne pioche pas simplement des algos sur des sites déjà existants, sans quoi ce site n'aurait aucun intérêt. Tous mes algos sont retravaillés avec l'objectif d'avoir le meilleur algo possible pour chaque cas (c'est subjectif bien évidemment, mais tout de même). Pour les sets incomplets, il est possible (et même fort probable) d'avoir des algos vraiment nuls, c'est que ces algos n'ont pas encore été retravaillés, et c'est pour ça que le set est affiché comme "incomplet" (et j'afficherai le set comme "complet" uniquement lorsque tous les algos auront été retravaillés. Exception : les OLL de mon site sont ceux que j'ai appris il y a plusieurs années, il est possible d'avoir un algo meilleur pour certains cas, mais que je ne connais pas.

Ce site est donc composé de :
  • Un nombre indéterminé de pages de sets d'algos
  • La page d'accueil, à partir de laquelle on peut tout retrouver
  • La page d'informations générales : c'est la page sur laquelle vous vous trouvez actuellement
  • La page de mes records perso
  • La page de ma collection de cubes
  • Le lien vers mon Drive, sur lequel se trouve des fichiers intéressants à propos du cube (pas mal de sets d'algos de Jabari, Brooks, etc, mais pas que)
  • Le lien d'invitation du discord francophone
  • Le lien direct pour m'envoyer un MP sur le forum francocube
  • La page de mes algos en cours : c'est en quelque sorte ma "zone de travail" pour le set que je suis en train d'apprendre, j'y dépose des algos, je retravaille, je compare, et quand je suis satisfait les algos choisis vont dans leur page de set
  • Le lien vers la liste UNR de France : c'est moi aussi qui gère la liste UNR, n'hésitez pas à me contacter pour tout nouvel ajout ou proposition
S'il y a d'autres questions, n'hésitez pas à me les poser via le discord ou le forum Francocube.

Bon cube à tous ! :D
