Les CLS du kilominx d'un Manchot

Les cas seront classés selon 7 catégories, qui sont les mêmes que pour les CLS de la méthode MGLS :

  • Set C : Le coin est déjà inséré correctement (OLL)
  • Set I+ : Le coin est mal inséré (orientation +)
  • Set I- : Le coin est mal inséré (orientation -)
  • Set O : Le coin n'est pas inséré et il est orienté correctement
  • Set + : Le coin n'est pas inséré et il est mal orienté (orientation +)
  • Set - : Le coin n'est pas inséré et il est mal orienté (orientation -)

L'orientation d'un coin est définie comme suit :

  • Un coin est bien orienté si et seulement si le sticker de la couleur du LL ou de la face opposée au LL pointe vers le haut ou vers le bas.
  • Si un coin est mal orienté, il est orienté + si une rotation de 120° dans le sens horaire l'orienterait, et il est orienté - si une rotation de 120° dans le sens anti-horaire l'orienterait.
